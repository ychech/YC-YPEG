# 更新日志

## v1.1.0 (2026-04-13)

### ✨ 特性与优化

- **首页视觉重构**：将底部 6 个功能卡片重构为“大图标 + 标题 + 描述”的布局，并统一更换为专属的“系列小狐狸”图标。
- **全局图标回归**：站点 Logo 和 Favicon 全面恢复为专属的“钻石”图标。
- **提示词指南新增「引言」**：在提示词工程指南中新增了独立的「引言 (Introduction)」页面。
- **引言页专属排版**：为引言页打造了无边框、无干扰（隐藏右侧大纲与底部翻页线）的极简沉浸式阅读体验，并为英文标题加入了倾斜与 3D 艺术字投影效果。
- **默认语言调整**：将站点的默认语言设置为中文（`zh`），自动适配中文用户的访问路径。

### 🐛 修复

- **路由 404 修复**：去除了导航栏和首页链接中多余的 `/zh/` 语言前缀，彻底解决了点击指南页面出现 `404` 的问题。
- **主题状态持久化**：修复了暗黑模式/亮色模式主题切换后状态无法持久化保存（刷新后丢失）的 Bug。

### 🗑️ 移除

- **废弃内容清理**：全局移除了不再维护的旧文档与导航目录。
