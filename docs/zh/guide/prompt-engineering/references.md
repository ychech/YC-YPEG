# 参考资料

1. **OpenAI Cookbook 相关资源汇总**
   - 网址：https://developers.openai.com/cookbook/articles/related_resources

2. **Brex Prompt Engineering**
   - **背景**：Brex 内部工程团队，用于快速赋能 600+ 员工构建 GPT-4 等 LLM 应用。
   - **开源地址**：https://github.com/brexhq/prompt-engineering
   - **定位**：生产级、实战导向，而非纯理论；强调 “可复现、可程序解析、可防御攻击”。
   - **核心比喻**：Prompt = Source Code，LLM = 非确定性解释器。

3. **Prompt Engineering Guide (社区最大知识库)**
   - 网址：https://www.promptingguide.ai
   - 特点：开源、免费，从基础到高级，含 COSTAR、RISEN 等框架。
   - 中文版：https://www.promptingguide.ai/zh

4. **AI Short (中文提示词大全)**
   - 网址：https://www.aishort.top
   - 特点：中文友好，分类清晰（写作、编程、办公等），可直接复制。

5. **吴恩达提示词工程系列课程**
   - 英文原版：https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/
   - 中文翻译：https://datawhalechina.github.io/llm-cookbook/#/C1/readme

6. **AI 导航站**
   - **猫目 AI 导航**：https://maomu.com/category/29 (收录了 2025 年最好的 602 个提示词工程工具)
   - **MeoAI 导航**：https://www.meoai.net/sitetag/免费

7. **Claude 提示词指南**
   - Prompt Engineering with Claude v3.X：https://catalog.us-east-1.prod.workshops.aws/workshops/0644c9e9-5b82-45f2-8835-3b5aa30b1848/en-US/lessons

8. **Auto-CoT 论文与代码**
   - **标题**：Auto-CoT: Automatic Chain of Thought Prompting in Large Language Models
   - **作者**：Zhuosheng Zhang, Aston Zhang, Mu Li, Alex Smola (Amazon Science)
   - **发表会议**：ICLR 2023（顶会）
   - **arXiv 链接**：https://arxiv.org/pdf/2210.03493
   - **代码地址**：https://github.com/amazon-science/auto-cot

## 实战建议

- 优先参考“官方 + 可复现”的资料。
- 对任何看起来过于具体的结论做交叉验证（多来源/多模型）。
- 把你自己的评估样本与结果沉淀下来，比收藏链接更有价值。
