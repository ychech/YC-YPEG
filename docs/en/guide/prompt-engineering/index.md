# YC Prompt Guide

This series of guides will help you systematically master the core concepts and practical techniques of Prompt Engineering.

## What is Prompt Engineering?

Prompt Engineering is a relatively new discipline focused on prompt development and optimization, helping users leverage Large Language Models (LLMs) for various scenarios and research fields.

## Guide Directory

- [Text Generation Parameters](./text-generation-params.md) - Deep dive into LLM text generation control parameters like Temperature, Top_p

## Quick Reference

:::tip Core Parameters
- **Temperature**: Controls randomness. Higher = more diverse.
- **Top_p**: Nucleus sampling. Controls candidate word range.
- **Tip**: Adjust only one parameter, keep the other at default 1.0.
:::
